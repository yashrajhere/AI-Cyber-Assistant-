# CyberTerm-AI

CyberTerm-AI is a terminal-based cybersecurity chatbot built with Python.

It allows users to interact with a command-line AI assistant that answers basic cybersecurity questions and provides a hacker-style CLI interface.

## Features

- Terminal based chatbot
- Cybersecurity knowledge responses
- Hacker-style banner UI
- Simple command system
- Additional Features for CyberTerm-AI
1️⃣ Interactive Command Line Chat
Users can interact with the chatbot directly from the terminal using natural language queries related to cybersecurity topics.
2️⃣ Cybersecurity Knowledge Base
The chatbot contains built-in responses for common security topics such as:
SQL Injection
Cross-Site Scripting (XSS)
Malware
Distributed Denial of Service (DDoS)
This helps beginners quickly understand security concepts.
3️⃣ Hacker-Style Terminal Interface
The tool features a visually appealing CLI interface with a colored ASCII banner and structured command prompts, creating a professional terminal experience.
4️⃣ Command System
The chatbot supports built-in commands for easy navigation:
help  – display available commands
clear – clear terminal screen
exit  – terminate chatbot session
5️⃣ Lightweight & Fast
The chatbot runs locally using Python, requiring minimal system resources and no heavy AI infrastructure.
6️⃣ Beginner-Friendly Cyber Assistant
The tool is designed as an educational cybersecurity assistant that helps users quickly learn security concepts without leaving the terminal.
7️⃣ Modular Architecture
The project is divided into multiple modules:
main.py – chatbot interface
chatbot.py – response logic
requirements.txt – dependencies
This structure makes the project easy to extend and maintain.
8️⃣ Extensible Design
Developers can easily add:
more cybersecurity topics
API integrations
AI models
automation commands

## Run

pip install -r requirements.txt

python main.py
